Installation av Fluigent SDK för Python
Författad av Sigge Axelsson – 2025-08-15

---

1. Hämta SDK från GitHub
   Om du redan har tillgång till filerna finns en länk i instruktionsmanualen som tillhör sensorn (Fluigent SDK.pdf).
   Annars kan du använda den publika länken:
   https://github.com/Fluigent/fgt-SDK/releases

Från GitHub, ladda ner versionen fgt-SDK-23.0.0 (eller den senaste tillgängliga versionen).

2. Navigera till Python-paketet
   Efter nedladdning, öppna den hämtade SDK-mappen och gå igenom undermapparna tills du hittar mappen "Python".
   I denna mapp finns en ZIP-fil med namnet fluigent\_sdk-23.0.0.zip (namnet kan variera om en nyare version finns).

3. Installera paketet via pip
   Öppna din IDE (t.ex. Visual Studio Code) eller kommandotolken (CMD), och kör:
   pip install fluigent_sdk-23.0.0.zip

OBS: Detta fungerar endast om du befinner dig i samma mapp som ZIP-filen.
Annars måste du först navigera dit med kommandot cd, öppna pp mappen i ditt IDE eller installera paketet genom att ange den fullständiga sökvägen, exempelvis:
python -m pip install "C:\Users\sigge.axelsson\OneDrive - Mesa Labs\Documents\Testrigg\_Pumpar\Kommunikation till Raspb\fgt-SDK-23.0.0\SDK-23.0.0\Python\fluigent_sdk-23.0.0.zip"

För mer information om pip-installationer finns det gott om guider online.

4. Installera beroendet setuptools
   Innan du kan köra Python-exemplen behöver du även installera paketet setuptools, annars får du felmeddelandet:
   ModuleNotFoundError: No module named 'pkg\_resources'

Installera det med:
pip install setuptools

5. Klart!
   Nu kan du köra Python-exemplen som medföljer SDK\:t.
